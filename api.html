API Documentation¶

This is the full API documentation of the unbalanced_dataset toolbox.

Under-sampling methods¶

The unbalanced_dataset.under_sampling provides methods to under-sample a dataset.

Classes¶

unbalanced_dataset.under_sampling.ClusterCentroids([...]) Class to perform under-sampling by generating centroids base on clustering.
unbalanced_dataset.under_sampling.CondensedNearestNeighbour([...]) Class to perform under-sampling based on the condensed nearest neighbour method.
unbalanced_dataset.under_sampling.EditedNearestNeighbours([...]) Class to perform under-sampling based on the edited nearest neighbour method.
unbalanced_dataset.under_sampling.InstanceHardnessThreshold(...) Class to perform under-sampling based on the instance hardness threshold.
unbalanced_dataset.under_sampling.NearMiss([...]) Class to perform under-sampling based on NearMiss methods.
unbalanced_dataset.under_sampling.NeighbourhoodCleaningRule([...]) Class performing under-sampling based on the neighbourhood cleaning rule.
unbalanced_dataset.under_sampling.OneSidedSelection([...]) Class to perform under-sampling based on one-sided selection method.
unbalanced_dataset.under_sampling.RandomUnderSampler([...]) Class to perform random under-sampling.
unbalanced_dataset.under_sampling.TomekLinks([...]) Class to perform under-sampling by removing Tomek’s links.

Over-sampling methods¶

The unbalanced_dataset.over_sampling provides a set of method to perform over-sampling.

Classes¶

unbalanced_dataset.over_sampling.RandomOverSampler([...]) Class to perform random over-sampling.
unbalanced_dataset.over_sampling.SMOTE([...]) Class to perform over-sampling using SMOTE.

Combination of over- and under-sampling methods¶

The unbalanced_dataset.combine provides methods which combine over-sampling and under-sampling.

Classes¶

unbalanced_dataset.combine.SMOTEENN([ratio, ...]) Class to perform over-sampling using SMOTE and cleaning using ENN.
unbalanced_dataset.combine.SMOTETomek([...]) Class to perform over-sampling using SMOTE and cleaning using Tomek links.

Ensemble methods¶

The unbalanced_dataset.ensemble module include methods generating under-sampled subsets combined inside an ensemble.

Classes¶

unbalanced_dataset.ensemble.BalanceCascade([...]) Perform under-sampling using an ensemble of subset selected using some classifiers.
unbalanced_dataset.ensemble.EasyEnsemble([...]) Perform under-sampling using an ensemble of random selection.
