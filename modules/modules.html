API Reference¶

unbalanced_dataset.under_sampling: Methods performing under-sampling¶

The unbalanced_dataset.under_sampling provides methods to under-sample a dataset.

Classes:

ClusterCentroids([ratio, random_state, ...]) Class to perform under-sampling by generating centroids base on clustering.
CondensedNearestNeighbour([return_indices, ...]) Class to perform under-sampling based on the condensed nearest neighbour method.
EditedNearestNeighbours([return_indices, ...]) Class to perform under-sampling based on the condensed nearest neighbour method.
NearMiss([ratio, return_indices, ...]) Class to perform under-sampling based on NearMiss methods.
NeighbourhoodCleaningRule([return_indices, ...]) Class performing under-sampling based on the neighbourhood cleaning rule.
OneSidedSelection([return_indices, ...]) Class to perform under-sampling based on one-sided selection method.
RandomUnderSampler([ratio, return_indices, ...]) Class to perform random under-sampling.
TomekLinks([return_indices, random_state, ...]) Class to perform under-sampling by removing Tomek’s links.

unbalanced_dataset.over_sampling: Methods performing over-sampling¶

The unbalanced_dataset.over_sampling provides a set of method to perform over-sampling.

Classes:

RandomOverSampler([ratio, random_state, verbose]) Class to perform random over-sampling.
SMOTE([ratio, random_state, verbose, k, m, ...]) Class to perform over-sampling using SMOTE.

unbalanced_dataset.under_sampling: Methods performing under-sampling¶

The unbalanced_dataset.ensemble module include methods generating under-sampled subsets combined inside an ensemble.

Classes:

BalanceCascade([ratio, return_indices, ...]) Perform under-sampling using an ensemble of subset selected using some classifiers.
EasyEnsemble([ratio, return_indices, ...]) Perform under-sampling using an ensemble of random selection.

unbalanced_dataset.combine: Methods combining over- and under-sampling¶

The unbalanced_dataset.combine provides methods which combine over-sampling and under-sampling.

Classes:

SMOTEENN([ratio, random_state, verbose, k, ...]) Class to perform over-sampling using SMOTE and cleaning using ENN.
SMOTETomek([ratio, random_state, verbose, ...]) Class to perform over-sampling using SMOTE and cleaning using Tomek links.
