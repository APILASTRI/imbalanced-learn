Module contents¶

Toolbox for unbalanced dataset in machine learning.

UnbalancedDataset is a set of python methods to deal with unbalanced datset in machine learning and pattern recognition.

Subpackages¶

combine
Module which provides methods based on over-sampling and under-sampling.
ensemble
Module which provides methods generating an ensemble of under-sampled subsets.
over_sampling
Module which provides methods to under-sample a dataset.
under-sampling
Module which provides methods to over-sample a dataset.
utils
Module which provides helper methods.
